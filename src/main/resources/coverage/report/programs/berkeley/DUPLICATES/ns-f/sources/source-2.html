34 /*
35  * Base class for all JE concrete MBeans. 
36  *
37  * It implements the MBeanRegistrar interface and defines the common part of
38  * those concrete MBeans.
39  *
40  * It defines the abstract methods which must be implemented in concrete
41  * MBeans.
42  *
43  * Subclasses of JEMBean must avoid passing JE exceptions across the network to
44  * the client side, since the client side will not have the proper JE exception
45  * class. For example, a method which does this:
46  *  
47  *  catch (DatabaseException databaseEx) {
48  *     new MBeanException(databaseEx);
49  *  }
50  *
51  * will result in a ClassNotFoundException on the client side when it receives
52  * the MBeanException, since it does not have DatabaseException.
53  */
