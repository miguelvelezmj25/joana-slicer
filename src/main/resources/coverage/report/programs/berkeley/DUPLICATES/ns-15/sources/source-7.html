19 /**
20  * Thrown when a lock has been "stolen", or preempted, from a transaction in a

23  * <p>The {@link com.sleepycat.je.Transaction} handle is invalidated as a
24  * result of this exception.</p>
25  * 
26  * <p>Locks may be preempted in a JE HA environment on a Replica system when
27  * the HA write operation needs a lock that an application reader transaction

30  *
31  * <p>Normally, applications should catch the base class {@link

35  * information.</p>

38  */
