25 /**
26  * This class writes out a log entry that can be used for replication syncup.
27  * It can be issued arbitrarily by the master at any point, in order to bound
28  * the syncup interval in much the way that a checkpoint bounds the recovery
29  * interval. The entry will a replicated one, which means that it will be
30  * tagged with a VLSN.
31  *
32  * Although this is a replication class, it resides in the utilint package
33  * because it is referenced in LogEntryType.java.
34  *
35  * TODO: This is currently not used. When it is used, it will be the first
36  * replicated log entry that does not have a real txn id. All replicated
37  * entries are expected to have negative ids, and the matchpoint should be
38  * exempt from Replay.updateSequences, or it should pass in a special reserved
39  * negative id, so as not to incur the assertion in Replay.updateSequences,
40  * that the txn id is <0.
41  */
