33  * A visitor to visit a Java class. The methods of this class must be called in
34  * the following order: <tt>visit</tt> [ <tt>visitSource</tt> ] [
35  * <tt>visitOuterClass</tt> ] ( <tt>visitAnnotation</tt> |
36  * <tt>visitTypeAnnotation</tt> | <tt>visitAttribute</tt> )* (
37  * <tt>visitInnerClass</tt> | <tt>visitField</tt> | <tt>visitMethod</tt> )*
38  * <tt>visitEnd</tt>.
39  * 
40  * @author Eric Bruneton
41  */
