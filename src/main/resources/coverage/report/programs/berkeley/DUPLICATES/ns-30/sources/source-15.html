19  * Provides information about the file store associated with a specific file.
20  * This class is a wrapper for the information made available by the {@code
21  * java.nio.file.FileStore} class introduced in Java 7, but using reflection to
22  * permit Java 6 to determine cleanly at runtime that file stores are not
23  * supported.
24  *
25  * TODO:
26  * We no longer support Java 6 so much of this mechanism can be removed. Other
27  * code can assume that a FileStore is always available. Also, if an
28  * IOException is thrown by FileStore methods, instead of tolerating this we
