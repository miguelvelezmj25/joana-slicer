18 /**
19  * Exception that is thrown when a security failure has occurred which
20  * may terminate the current replication stream. When it is raised, the
21  * replication stream consumer is no longer eligible to consume the stream.
22  * <p>
23  *
24  * This exception covers following security failure during streaming:
25  * <ul>
26  * <li> The user attempted to contact the feeder of a secure store without
27  * authenticating. It will be raised during client does service handshake
28  * with server in this case;
29  *
30  * <li> There was a problem authenticating the stream client during stream, say
31  * because the token provided by client is no longer valid during streaming;
32  *
33  * <li> Stream client attempted to perform an operation that they were not
34  * authorized to perform, say, the stream client is trying to stream data
35  * from a table that she is not eligible to read.
36  *
37  * </ul>
38  * @hidden For internal use only
39  */
40 public class ReplicationSecurityException extends DatabaseException {
