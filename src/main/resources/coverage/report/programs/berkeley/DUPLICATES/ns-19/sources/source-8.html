28 /**
29  * Defines the request/response messages used in the implementation of
30  * elections.
31  *
32  * From Proposer to Acceptor:
33  *     Propose -> Promise | Reject
34  *     Accept -> Accepted | Reject
35  *
36  * From Proposer initiator to Learners:
37  *    Result -> none
38  *
39  * The following exchange is not part of the elections process, but is used by
40  * the Monitor to query a Learner for the latest election result it's aware of,
41  * when the Monitor first starts up.  It is also used by nodes and utilities
42  * that are attempting to find the master.
43  *
44  * From Monitor to Learner
45  *    MasterQuery -> MasterQueryResponse | None
46  *
47  */

122      * Promise response message. It's sent in response to a Propose message.
123      *
124      * Note that the "minor" part of the suggestion ranking is always tagged on
125      * to the end of the promise request payload. Older pre 7.1.3 nodes will
126      * ignore extra tokens at the end, since they do not know about the minor
127      * component of the ranking. This node will use it if it's present and
128      * otherwise use a value of zero for the minor (VLSN) component.
129      *
130      * So when comparing rankings across old and new nodes, we are effectively
131      * comparing a Ranking(VLSN, Long.MIN_VALUE) with a Ranking(DTVLSN, VLSN),
132      * resulting in suboptimal election results (from a dtvlsn perspective)
133      * while an upgrade is in progress, that is, it will tend to favor the
134      * older node. But this inaccuracy will vanish once all nodes have been
135      * upgraded.
136      */

167         public Promise(String responseLine, String[] tokens)
168             throws InvalidMessageException {
169 
170             super(responseLine, tokens);
171             highestProposal = proposalParser.parse(nextPayloadToken());
172             acceptedValue = valueParser.parse(nextPayloadToken());
173             suggestion = valueParser.parse(nextPayloadToken());
174             String weight = nextPayloadToken();
175             long majorRanking =
176                 "".equals(weight) ?
177                 Ranking.UNINITIALIZED.major :
178                 Long.parseLong(weight);
179             long minorRanking = Ranking.UNINITIALIZED.major;
180             priority = Integer.parseInt(nextPayloadToken());
181             if (getMajorVersionNumber(sendVersion) > 1) {
182                 logVersion = Integer.parseInt(nextPayloadToken());
183                 jeVersion = new JEVersion(nextPayloadToken());
184                 if (hasMoreTokens()) {
185                     /*

187                      * of JE nodes >= version 7.1.3
188                      */
189                     minorRanking = Long.parseLong(nextPayloadToken());
190                 }
191             }
192             suggestionRanking = new Ranking(majorRanking, minorRanking);
193         }
