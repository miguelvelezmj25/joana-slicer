38  * A List view of a {@link Database}.
39  *
40  * <p>For all stored lists the keys of the underlying Database
41  * must have record number format, and therefore the store or index must be a
42  * RECNO, RECNO-RENUMBER, QUEUE, or BTREE-RECNUM database.  Only RECNO-RENUMBER
43  * allows true list behavior where record numbers are renumbered following the
44  * position of an element that is added or removed.  For the other access
45  * methods (RECNO, QUEUE, and BTREE-RECNUM), stored Lists are most useful as
46  * read-only collections where record numbers are not required to be
47  * sequential.</p>
48  *
49  * <p>In addition to the standard List methods, this class provides the
50  * following methods for stored lists only.  Note that the use of these methods
51  * is not compatible with the standard Java collections interface.</p>
52  * <ul>
53  * <li>{@link #append(Object)}</li>
54  * </ul>
55  * @author Mark Hayes
56  */
