18 /**
19  * A simple <code>Map.Entry</code> implementation that can be used as in
20  * input parameter.  Since a <code>MapEntryParameter</code> is not obtained
21  * from a map, it is not attached to any map in particular.  To emphasize that
22  * changing this object does not change the map, the {@link #setValue} method
23  * always throws <code>UnsupportedOperationException</code>.
24  *
25  * <p><b>Warning:</b> Use of this interface violates the Java Collections
26  * interface contract since these state that <code>Map.Entry</code> objects
27  * should only be obtained from <code>Map.entrySet()</code> sets, while this
28  * class allows constructing them directly.  However, it is useful for
29  * performing operations on an entry set such as add(), contains(), etc.  For
30  * restrictions see {@link #getValue} and {@link #setValue}.</p>
31  *
32  * @author Mark Hayes
33  */
