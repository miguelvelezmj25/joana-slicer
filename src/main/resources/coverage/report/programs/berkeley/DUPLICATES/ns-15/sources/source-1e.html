33 /**
34  * Specifies the immutable attributes of a replicated environment.
35  * <p>
36  * To change the default settings for a replicated environment, an application
37  * creates a configuration object, customizes settings and uses it for {@link
38  * ReplicatedEnvironment} construction. The set methods of this class validate
39  * the configuration values when the method is invoked.  An
40  * IllegalArgumentException is thrown if the value is not valid for that
41  * attribute.
42  * <p>
43  * Note that ReplicationConfig only describes those attributes which must be
44  * set at {@code ReplicatedEnvironment} construction time, while its superclass
45  * {@link ReplicationMutableConfig} describes attributes that may be modified
46  * during the life of the replication group.
47  * <p>
48  * ReplicationConfig follows precedence rules similar to those of
49  * {@link EnvironmentConfig}.
50  * <ol>
51  * <li>Configuration parameters specified
52  * in {@literal <environmentHome>/je.properties} take first precedence.</li>
53  * <li>Configuration parameters set in the ReplicationConfig object used
54  * at {@code ReplicatedEnvironment} construction are next.</li>
55  * <li>Any configuration parameters not set by the application are set to
56  * system defaults, described along with the parameter name String constants
57  * in this class.</li>
58  *</ol>
59  * <p>
60  * After a {@code ReplicatedEnvironment} has been constructed, its mutable
61  * properties may be changed using {@code
62  * ReplicatedEnvironment#setMutableConfig}.  See {@code
63  * ReplicationMutableConfig} for a list of mutable properties; all other
64  * properties are immutable.  Whether a property is mutable or immutable is
65  * also described along with the parameter name String constants in this class.
66  *
67  * <h4>Getting the Current ReplicatedEnvironment Properties</h4>
68  *
69  * To get the current "live" properties of a replicated environment after
70  * constructing it or changing its properties, you must call {@link
71  * ReplicatedEnvironment#getRepConfig} or {@link
72  * ReplicatedEnvironment#getRepMutableConfig}.  The original ReplicationConfig
73  * or ReplicationMutableConfig object used to set the properties is not kept up
74  * to date as properties are changed, and does not reflect property validation
75  * or properties that are computed.
76  */
