33 /**
34  * @hidden SSL deferred
35  * This is the root class for specifying the parameters that control
36  * replication network communication within a a replicated environment. The
37  * parameters contained here are immutable.
38  * <p>
39  * To change the default settings for a replicated environment, an application
40  * creates a configuration object, customizes settings and uses it for {@link
41  * ReplicatedEnvironment} construction. Except as noted, the set methods of
42  * this class perform only minimal validation of configuration values when the
43  * method is called, and value checking is deferred until the time a
44  * DataChannel factory is constructed. An IllegalArgumentException is thrown
45  * if the value is not valid for that attribute.
46  * <p>
47  * ReplicationNetworkConfig follows precedence rules similar to those of
48  * {@link EnvironmentConfig}.
49  * <ol>
50  * <li>Configuration parameters specified
51  * in {@literal <environmentHome>/je.properties} take first precedence.</li>
52  * <li>Configuration parameters set in the ReplicationNetworkConfig object used
53  * at {@code ReplicatedEnvironment} construction are next.</li>
54  * <li>Any configuration parameters not set by the application are set to
55  * system defaults, described along with the parameter name String constants
56  * in this class.</li>
57  *</ol>
58  * <p>
59  *
