32 /**
33  * The base class for classes that provide entity model metadata.  An {@link
34  * EntityModel} defines entity classes, primary keys, secondary keys, and
35  * relationships between entities.  For each entity class that is part of the
36  * model, a single {@link PrimaryIndex} object and zero or more {@link
37  * SecondaryIndex} objects may be accessed via an {@link EntityStore}.
38  *
39  * <p>The built-in entity model, the {@link AnnotationModel}, is based on
40  * annotations that are added to entity classes and their key fields.
41  * Annotations are used in the examples in this package, and it is expected
42  * that annotations will normally be used; most readers should therefore skip
43  * to the {@link AnnotationModel} class.  However, a custom entity model class
44  * may define its own metadata.  This can be used to define entity classes and
45  * keys using mechanisms other than annotations.</p>
46  *
47  * <p>A concrete entity model class should extend this class and implement the
48  * {@link #getClassMetadata}, {@link #getEntityMetadata} and {@link
49  * #getKnownClasses} methods.</p>
50  *
51  * <p>This is an abstract class rather than an interface to allow adding
52  * capabilities to the model at a future date without causing
53  * incompatibilities.  For example, a method may be added in the future for
54  * returning new information about the model and subclasses may override this
55  * method to return the new information.  Any new methods will have default
56  * implementations that return default values, and the use of the new
57  * information will be optional.</p>
58  *
59  * @author Mark Hayes
60  */
61 public abstract class EntityModel {

92      * Registers a persistent class, most importantly, a {@link
93      * PersistentProxy} class or entity subclass. Also registers an enum or 
94      * array class.
95      *
96      * <p>Any persistent class , enum class or array may be registered in 
97      * advance of using it, to avoid the overhead of updating the catalog 
98      * database when an instance of the class is first stored.  This method 
99      * <em>must</em> be called in three cases:</p>
100      * <ol>
101      * <li>to register all {@link PersistentProxy} classes, and</li>
102      * <li>to register an entity subclass defining a secondary key, if {@link
103      * EntityStore#getSubclassIndex getSubclassIndex} is not called for the
104      * subclass, and</li>
105      * <li>to register all new enum or array classes, if the these enum or 
106      * array classes are unknown for DPL but will be used in a Converter
107      * mutation.
108      * </li>
109      * </ol>
110      *
111      * <p>For example:</p>
112      *
113      * <pre class="code">
114      * EntityModel model = new AnnotationModel();
115      * model.registerClass(MyProxy.class);
116      * model.registerClass(MyEntitySubclass.class);
117      * model.registerClass(MyEnum.class);
118      * model.registerClass(MyArray[].class);
119      *
120      * StoreConfig config = new StoreConfig();
121      * ...
122      * config.setModel(model);
123      *
124      * EntityStore store = new EntityStore(..., config);</pre>
125      *
126      * <p>This method must be called before opening a store based on this
127      * model.</p>
128      *
129      * @param persistentClass the class to register.
130      *
131      * @throws IllegalStateException if this method is called for a model that
132      * is associated with an open store.
133      *
134      * @throws IllegalArgumentException if the given class is not persistent
135      * or has a different class loader than previously registered classes.
136      */
137     public final void registerClass(Class persistentClass) {
138         if (catalog != null) {
139             throw new IllegalStateException("Store is already open");
140         } else {
141             String className = persistentClass.getName();
142             ClassMetadata meta = getClassMetadata(className);
143             if (meta == null &&
144                 !persistentClass.isEnum() &&
145                 !persistentClass.isArray()) {
146                 throw new IllegalArgumentException
147                     ("Class is not persistent, or is not an enum or array: " + 
148                      className);
149             }
150         }
151     }
