27 /**
28  * A Log entry allows you to read, write and dump a database log entry.  Each
29  * entry may be made up of one or more loggable items.
30  *
31  * The log entry on disk consists of
32  *  a. a log header defined by LogManager
33  *  b. a VLSN, if this entry type requires it, and replication is on.
34  *  c. the specific contents of the log entry.
35  *
36  * This class encompasses (b and c).
37  *
38  * @param <T> the type of the loggable items in this entry
39  */
